# Contact Manager

This is an **ASP.NET** web application for managing contacts, built using **ASP.NET Razor Pages**, **Entity Framework Core**, and **MySQL**.

## Tech Stack
- ASP.NET Razor Pages  
- Entity Framework Core  
- MySQL  
- .NET 9  

## Notes on .NET Version and Deployment
